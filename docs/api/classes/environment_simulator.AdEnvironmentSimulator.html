    Class AdEnvironmentSimulator

    Offline ad environment simulator.

    Produces synthetic performance signals per platform and computes reward with margin-ROAS shaping and overspend penalties. Supports constraining platforms, creatives, and budget clamps to mirror real-world constraints.
